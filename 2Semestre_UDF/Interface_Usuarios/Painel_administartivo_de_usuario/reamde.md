# 🎓 Sistema de Gestão Acadêmica Mobile

Este projeto apresenta um protótipo de alta fidelidade para uma plataforma de gestão universitária, focado na experiência do estudante (Student Experience - SX). O objetivo é centralizar a vida acadêmica em uma interface móvel que prioriza a agilidade e a clareza visual.

## 🚀 O Desafio

Estudantes universitários lidam com uma sobrecarga de informações dispersas em diferentes portais (notas, faltas, materiais, calendários). O desafio deste projeto foi projetar uma interface que reduzisse a carga cognitiva, permitindo que o aluno encontre o que precisa em menos de três cliques.

## 🎨 Design System & Identidade Visual

A interface foi construída do zero, focando em consistência e escalabilidade:

### 🌈 Paleta de Cores
- **Cor Primária:** Azul institucional, transmitindo confiança e profissionalismo.
- **Cores de Suporte:** Tons de cinza neutro para superfícies e cores semânticas (verde para aprovação, vermelho para prazos críticos).

### 🖋️ Tipografia
- Foi utilizada uma tipografia sem serifa de alta legibilidade, com escalas hierárquicas bem definidas para destacar títulos e informações de status.

### 🍱 Componentização
- **Atomic Design:** Desenvolvimento baseado em átomos e moléculas (botões, inputs, cards).
- **Auto-Layout:** Todas as telas utilizam auto-layout no Figma para garantir responsividade e alinhamento preciso.
- **Variantes:** Uso de componentes variantes para estados de botões (hover, active, disabled) e ícones.

## 📱 Fluxo de Navegação

O protótipo cobre os fluxos principais da jornada do estudante:

1.  **Onboarding & Login:** Acesso seguro e simplificado.
2.  **Dashboard (Home):** Visão geral de horários do dia, notificações urgentes e progresso do semestre.
3.  **Módulo de Disciplinas:** Detalhamento de cada matéria, incluindo histórico de notas e frequência.
4.  **Agenda & Prazos:** Calendário interativo com visualização por lista de tarefas.
5.  **Central de Relatórios:** Gráficos de desempenho para acompanhamento da evolução acadêmica.

## 🛠️ Ferramentas Utilizadas

- **Figma:** Design de interface, prototipagem de interações e micro-animações.
- **Iconify/Phosphor Icons:** Biblioteca de ícones padronizada.
- **Princípios de UX Research:** Mapeamento de dores do usuário para definição de funcionalidades.
